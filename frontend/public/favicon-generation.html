Favicon Generation Instructions

To generate proper favicon files for AFRI-CABINDA, follow these steps:

Step 1: Prepare Your Logo

Use the shop-logo.jpg file in the public folder as your source image.

Step 2: Generate Favicon Files

Visit one of these online tools:

  • favicon.io
  • RealFaviconGenerator.net
  • Favicon Generator

Step 3: Required Files

Generate and place these files in the public folder:

  • favicon.ico (16x16, 32x32)
  • favicon-16x16.png
  • favicon-32x32.png
  • apple-touch-icon.png (180x180)

Step 4: Update HTML

The HTML file already includes the proper favicon links. Just replace the placeholder files with your generated ones.

Step 5: Test

After placing the favicon files, test them by:

  • Refreshing your browser
  • Checking the browser tab for the favicon
  • Testing on mobile devices
